这是[宁皓网](http://ninghao.net)的 webpack 工具演示课程里使用的代码。使用 webpack 打包 React 项目，在项目里你可以使用 es2015 的写法，可以使用 JSX 。还有模块热替换功能，就是你编辑了项目文件以后，不会刷新整个页面，而是只替换有变化的地方。

1 → 克隆仓库到本地
```
git clone https://github.com/ninghao/ninghao-webpack.git
```
2 → 安装项目需要的东西
```
cd ninghao-webpack
npm install
```
3 → 运行开发服务器
```
npm run watch
```
4 → 在浏览器里打开
```
http://localhost:8080
```
